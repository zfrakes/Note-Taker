# Note-Taker
A note taker application is able to create, save, and delete a user notes. This application uses an express backend and saves and retrieves note data from a JSON file.

## Technologies/Languages used

* HTML
* CSS
* JQuery
* db.json
* JavaScript
* Node
* api
